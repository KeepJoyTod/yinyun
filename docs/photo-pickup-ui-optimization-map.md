# 客户取片 UI 优化地图

更新时间：2026-06-11

## 当前结论

客户取片端已经从临时测试页升级为可给客户试用的移动端取片入口。H5、微信小程序、抖音小程序共用 `mobile-uniapp`，本轮改动会同时覆盖三端。

当前主链路：

```text
手机号 + 取片码 -> 相册列表 -> 照片目录 -> 照片预览 -> 下载/保存
```

正式 API 域名：

```text
https://api.evanshine.me/client/photo/*
```

## 设计原则

- 页面先服务客户取片，不展示服务器、OSS、后台、端口等技术信息。
- 图片长期地址不进入客户端，客户端只拿短期签名 URL 或后端 `/stream`。
- UI 以“清爽、可信、适合摄影交付”为主，不做营销页式大面积装饰。
