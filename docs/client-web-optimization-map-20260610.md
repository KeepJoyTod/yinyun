# 影约云客户电脑网页端优化地图

更新时间：2026-06-11

## 当前状态

`client-web` 已创建，端口 `5200`，当前提供：

- 官网首页 `/`
- 小程序预约引导 `/booking`
- 客户取片登录 `/customer/login`
- 客户相册列表 `/customer/albums`
- 客户相册详情 `/customer/albums/:albumId`
- 客户取片结果页 `/customer/result`
- 门店入口 `/staff`
- `/booking/success` 旧链接兼容重定向到 `/booking`
- 客户电脑网页不再调用 `/client/booking/intent`，预约下单只走微信/抖音小程序
